chemlift |release| Documentation
================================

chemlift is a Python package that provides tools for language-interfaced predictions using large language models-with a focus on chemistry and materials science applications. 
It is built on top of the gptchem package that provides these functionalities using the OpenAI API.
In addition to that, chemlift allows for using this approaches parameter efficient fine-tuning (PEFT) [PEFT]_ of open-source language models
